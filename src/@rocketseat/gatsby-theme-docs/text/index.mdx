[Botmation](https://github.com/mrWh1te/Botmation) is a TypeScript framework for declaratively building bots with [Puppeteer](https://github.com/puppeteer/puppeteer). It maximizes code *readability*, *maintainability* and *testability* with a [compositional design](https://en.wikipedia.org/wiki/Function_composition_%28computer_science%29).

These bots can do whatever people do with a web browser and more. They can surf and scrape complex web applications, automate social media, take screenshots, generate website PDF's and much more! 🌊🏄📸
