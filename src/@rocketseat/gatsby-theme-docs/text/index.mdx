[Botmation](https://github.com/mrWh1te/Botmation) is a simple TypeScript framework for [Puppeteer](https://github.com/puppeteer/puppeteer) to declaratively build web bots. It reduces code reusing barriers with *composition* (inspired by functional programming) to maximize code *readability*, *maintainability* and *testability*.
