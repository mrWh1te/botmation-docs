[Botmation](https://github.com/mrWh1te/Botmation) is a simple & declarative framework for building web bots with TypeScript.

It's built with [Puppeteer](https://github.com/puppeteer/puppeteer), a NodeJS library that provides a high-level API to control Chrome or Chromium over the [DevTools Protocol](https://chromedevtools.github.io/devtools-protocol/). You can use Puppeteer to build web bots to control the browser to handle various tasks like automate your social media, take screenshots & generate PDF's of websites, surf the web and more. 🌊🏄

[Botmation](https://github.com/mrWh1te/Botmation) gives you a simple, scalable, declarative framework to increase your code value. Botmation reduces code reusing barriers with *composition* and maximizes your code's *readability*, *maintainability* and *testability*.

## Features

- 🤓 Learn only what you need
- 🛣️ Simple pattern to follow
- 🥇 100% Library Test Coverage
- 📚 Get started with Functional Programming
- ⚡️ Write readable, testable & composable code
- 🏢 Use higher order functions
- 🧱 Assemble complex async functionality
- and much more 🔥
