In the `./src/examples` [directory](https://github.com/mrWh1te/Botmation/tree/master/src/examples) of Botmation's repo, exists a small collection of simple bots, to help you get started:

- [Getting Started: Object-Oriented](https://github.com/mrWh1te/Botmation/tree/master/src/examples/simple_objectoriented.ts)
- [Getting Started: Functional](https://github.com/mrWh1te/Botmation/tree/master/src/examples/simple_functional.ts)
- [Taking Screenshots](https://github.com/mrWh1te/Botmation/tree/master/src/examples/screenshots.ts)
- [Logging into Instagram](https://github.com/mrWh1te/Botmation/tree/master/src/examples/instagram.ts)

You can copy/paste the code or setup Botmation's repo locally to run the Bots from there.
